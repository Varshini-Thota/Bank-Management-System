# Bank Management System

## Overview

A menu-driven Bank Management System developed in C. The project allows users to create an account, deposit money, withdraw money, and check account balance.

## Features

* Create Account
* Deposit Money
* Withdraw Money
* Check Balance
* Menu Driven Interface

## Technologies Used

* C Programming
* OnlineGDB

## Applications

* Banking System Simulation
* Learning File Handling and Structures
* C Programming Practice
